## 知識の網羅性

- [ ] Given テキスト抽出が必要な時
      When ガイドを参照する
      Then markitdown による変換方法が得られる

- [ ] Given 新規プレゼンを作成する時
      When pptxgenjs リファレンスを参照する
      Then pptxgenjs によるスライド構築のコード例（テキスト、チャート、画像、表、図形）が得られる

- [ ] Given 既存テンプレートを編集する時
      When 編集リファレンスを参照する
      Then 展開 → XML 編集 → クリーンアップ → 再パッキングのワークフローが得られる

- [ ] Given デザインガイドラインが必要な時
